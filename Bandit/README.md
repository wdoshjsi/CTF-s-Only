# 🚀 Bandit CTF Writeups (OverTheWire)

## 📌 Overview

This repository contains structured writeups for selected challenges from the **Bandit wargame** by OverTheWire. The focus is not just solving challenges, but understanding the underlying Linux concepts, tools, and thinking process behind each solution.

The goal of this repo is to demonstrate:

* Linux fundamentals
* Problem-solving approach
* Command-line proficiency
* Cybersecurity thinking process

---

## 🎯 Objectives

* Build strong foundational Linux skills
* Learn practical command-line tools used in cybersecurity
* Develop structured documentation habits
* Prepare for real-world penetration testing workflows

---

## 🧠 What This Repo Focuses On

Instead of only listing answers, each writeup includes:

* Problem breakdown
* Thinking process
* Commands used
* Key technical insights
* Final result

This is designed to reflect **how a cybersecurity learner thinks**, not just what they executed.

---

Each challenge demonstrates a different core Linux concept such as:

* File type detection
* File system searching
* Text processing
* Data filtering
* Encoding & decoding
